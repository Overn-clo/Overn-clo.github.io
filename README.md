# projet_stage_eric_site
Ce projet est le site vitrine d'un préparateur mental. L'objectif principal est de mettre en avant ses services et d'accompagner les utilisateurs ciblés (sportifs, étudiants) vers la performance et le bien-être.

Fonctionnalités :
Design Responsive : Site entièrement adapté aux mobiles et ordinateurs.

Présentation des services : Détail des programmes d'accompagnement.

Formulaire de contact : Système de prise de contact direct via PHP.

Optimisation SEO : Structure HTML sémantique pour un meilleur référencement.

Technologies utilisées
HTML5 : Structure du contenu.

CSS : Design, mise en page et animations (Media Queries pour le responsive).

JavaScript : Interactivité (menu burger, validation de formulaire, etc.).

PHP : Gestion du backend (envoi d'e-mails, logique serveur).

Installation locale
Pour faire tourner ce projet sur votre machine :

Cloner le dépôt :

Bash
git clone https://github.com/Overn-clo/Projet_stage_eric_site.git

Lancer un serveur local :
Comme le projet utilise du PHP, vous devez utiliser un serveur local type XAMPP, WAMP ou MAMP.

Déplacer le dossier dans le répertoire htdocs ou www de votre serveur.

Accéder au site via http://localhost/Projet_stage_eric_site.

Structure du projet :

Plaintext :

├── assets/
